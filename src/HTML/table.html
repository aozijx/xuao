PyInstaller 跨平台支持、自动处理依赖、打包为单个文件、适合大多数程序 文件较大、启动较慢、兼容性问题、调试困难
cx_Freeze 跨平台支持、定制性强、打包文件体积较小 配置复杂、依赖处理不如 PyInstaller 自动
py2exe 打包文件小、Windows 优化、配置简单 仅支持 Windows、对新版本 Python 支持差
PyOxidizer 体积小、嵌入 Python 解释器、WebAssembly 支持 配置复杂、文档较少、更新较慢

Terser Terser 是一个现代的 JavaScript 压缩工具，性能和功能上都非常好。 无 npm install terser --save-dev npx terser src/main.js --compress --mangle --output dist/main.min.js src/main.js 原始文件。 dist/main.min.js 压缩文件。 --compress 开启压缩。 --mangle 混淆变量名。 UglifyJS Online

CSSNano CSSNano 是一个非常流行的CSS压缩工具，能够优化和压缩 CSS 文件。 无 npm install cssnano --save-dev npx cssnano src/styles.css dist/styles.min.css src/styles.css是源文件。 dist/styles.min.css是压缩文件。 CSSMinifier

构建工具:Webpack Webpack是一种强大的模块打包工具，它不仅可以打包代码，还能通过 webpack.config.js配置自动化地压缩 JavaScript 和 CSS 文件。 在根目录下创建webpack.config.js配置文件，配置压缩插件 npm install --save-dev webpack webpack-cli terser-webpack-plugin css-minimizer-webpack-plugin npx webpack --config webpack.config.js 这将自动压缩JavaScript和CSS文件，并将结果输出到指定目录。
