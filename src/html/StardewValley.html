🐝 星露谷物语蜂房使用指南 🍯

✦ 蜂房基础 ✦

蜂房可以将花朵转化为各种有价值的蜂蜜！放置蜂房后，当半径5格范围内有开放的花朵时，每4天可以收获一次蜂蜜。

制作材料需求：

×40 木材
×8 煤炭
×1 铁锭
×1 枫糖浆

解锁条件： 耕种等级3级

产出：

加工时间：

6100 分 ≈ 4 天

✦ 使用步骤 ✦

  1. 建造蜂房
  2. 将蜂房放置在花朵附近
  3. 确保花朵与蜂房间无障碍物
  4. 等待4天收获蜂蜜

✦ 花朵对应蜂蜜类型 ✦

花朵 蜂蜜类型 售价
无/野花 野蜂蜜 100
蓝爵 蓝爵蜂蜜 200
向日葵 向日葵蜂蜜 260
夏季亮片 夏季亮片蜂蜜 280
虞美人 虞美人蜂蜜 380
玫瑰仙子 玫瑰仙子蜂蜜 680

✦ 专家建议 ✦

  • 冬季无法生产蜂蜜
  • 使用避雷针保护蜂房
  • 搭配工匠职业可提升售价
  • 蜂房可以放置在温室（不会产出蜂蜜）和姜岛

🥚 星露谷物语蛋黄酱机指南 🥚

✦ 基础信息 ✦

蛋黄酱机可将鸡蛋转化为不同品质的蛋黄酱！放入大鸡蛋可产出更高价值的蛋黄酱，铱星品质鸡蛋还能制作金色蛋黄酱。

制作材料：

×15 木材
×15 石头
×1 地精
×1 铜锭

解锁条件： 耕种等级2级

产出：

加工时间：

3 小时

✦ 使用流程 ✦

  1. 从鸡舍收集鸡蛋
  2. 将鸡蛋放入机器
  3. 等待3小时游戏时间
  4. 收取成品蛋黄酱

✦ 原料产出对照表 ✦

原料 成品 基础售价
普通鸡蛋 普通蛋黄酱 190
鸭蛋 鸭蛋黄酱 375
虚空蛋 虚空蛋黄酱 275
恐龙蛋 恐龙蛋黄酱 800

✦ 专家建议 ✦

  • 使用自动采集器可自动收集
  • 工匠职业可使售价提升40%
  • 金蛋可制作金色蛋黄酱（售价500g）
  • 蛋黄酱是完成献祭的重要物品

🍺 星露谷物语小桶指南 🍇

✦ 基础信息 ✦

小桶可将农产品加工为高价值商品！水果可酿造为酒类，蔬菜可制成腌菜，蜂蜜可制作蜜酒，是后期重要的收入来源。

制作材料：

×30 木材
×1 铜锭
×1 铁锭
×1 橡树树脂

解锁条件： 耕种等级8级

产出举例：

加工时间：

视种类而定

✦ 使用流程 ✦

  1. 收集水果/蔬菜/蜂蜜
  2. 将原料放入小桶
  3. 根据原料类型等待对应天数
  4. 收取加工成品

✦ 原料产出对照表 ✦

原料 成品 基础售价
杨桃 杨桃酒 2250
上古水果 上古果酒 1650
小麦 啤酒 200
蜂蜜 蜜酒 300

✦ 专家建议 ✦

  • 在棚屋内可密集摆放小桶
  • 选择基础价值高的水果收益更大
  • 使用加速剂可缩短酿造时间
  • 铱星品质原料不影响成品品质
